# alculadora-media-academica
Implementação de cálculos matemáticos convertendo tipos de dados (float) e utilizando estruturas condicionais (if/else) para verificar aprovação.

🧮 Calculadora de Médias Acadêmicas
Este projeto é uma aplicação desktop desenvolvida em Python para facilitar o cálculo de médias escolares ou universitárias. O que começou como um script simples de terminal evoluiu para uma ferramenta com interface gráfica moderna.

🚀 Evolução do Projeto
O objetivo inicial foi praticar lógica de programação e estruturas condicionais. Atualmente, o projeto conta com uma interface amigável para melhorar a experiência do usuário.

🛠️ Tecnologias Utilizadas
Python 3.x: Linguagem base.
CustomTkinter: Biblioteca utilizada para criar a interface gráfica (GUI) com um visual moderno e suporte a modo escuro/claro.
Logica de Programação: Tratamento de dados e cálculos matemáticos.

⚙️ Como rodar o projeto
1°Certifique-se de ter o Python instalado.
2°Instale a biblioteca necessária:
---> pip install customtkinter
3°Execute o arquivo principal:
---> python main.py
